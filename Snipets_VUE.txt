#Snipets - VUE

## Node, NVM, NPM, VITE, NPX
Node: Entorno que permite que javascript corra en el pc (requerido para usar vue)
npm: Node Package Manager: Gestor de paquetes y libreerias que necesita los proyectos vue
npx: Node Package Executor: Ejecuta herramientas sin instalarlas globalmente (crear proyecto de vue)
nvm: Node Version Manager: Gestor de versiones de node, que puede variar entre proyectos
vite: Herramienta para crear y ejecutar proyecto vue de forma veloz.

## Instalar VUE
1. Instalar nvm -> https://www.nvmnode.com/
2. nvm install --lts -> Instala node y npm latest version
3. (Opcional) Instalar vue-cli -> npm install -g @vue/cli (Para usar vue-cli)
4. (Opcional) Usar npx -> npx create-vue mi-proyecto (ejecutor de paquetes instala sin instalacion global)
5. (Opcional) Instala c/vite -> npm create vite@latest mi-proyecto / npx create-vite@latest

### Verificar paquetes
npm list -g --depth=0 -> Lista paquetes instalados globalmente

### NVM: Node version manager
Es un paquete que gestionas las versiones de node.js

### Crear y ejecutar programa con VUE-CLI
. vue create proyecto-nuevo
. npm run serve

### Crear y ejecutar programa con VITE
. npm create vite@latest proyecto-nuevo
. npm run dev

### Instalar dependencias de proyectos nuevo/existente
. npm install <dependencia> -> Instalar dependencias de proyecto (nuevo)
. npm install -> Instalar dependencias del proyecto (existente)

## Existen dos forma de usar vue:
- Option API (Vue 2 y 3)
`export default {
  data() {
    return {
      contador: 0
    }
  },
  methods: {
    sumar() {
      this.contador++;
    }
  }
}`
- Composition API (Vue 3) *Moderno
`<script setup>
	import { ref } from 'vue';
	const contador = ref(0);
	function sumar() {
		contador.value++;
	}
</script>`

## One-way y Two-way data bingind
- One-way: Informacion va unidireccional: js -> html
- Two-way: Informacion va bidireccional: js <-> html

'v-bind:class' | ':class': Vincular una variables dentro de un atributo html (one-way)
'{{ variable }}': Interpolacion de variables para mostrar datos en html (one-way)
'v-model='var': Une una variable de .js con un valor de html para que sea reactiva (two-way)

En Vue 3 se prefiere usar Arrow Function: const funcion = () => {...code};

En <script setup> todo debe ser variable, no usa this
