?. optional chaining: Evita errores de null o undefined en objetos

... -> Spread: Desempaquetar o exparcir 
a) Copiar o Unir dos arrary (const c = [...a, ...b])  
b) Clonar objetos (obj1 = obj2) si cambia uno cambia el otro, comparten memoria (copia superficial)
c) Argumentos a una funcion: Math.max(...numeros) // numeros = [1,2,3] -> 3

... -> Rest: Empaquetar o agrupar
a) Parametros infinitos en funciones: sumar = (...numeros) => {...code} // dentro de la funcion ahora numeros es un array de tamaño n
b) Destructuracion: Extraer variables de un array o objeto
const a = [a,b,c,d] -> const [1,2,...3] = a -> // a=1, b=2, c=[3,4]
const x = { a: "F", b: "M", c: 2, d: "R" } -> const {a, ...e} = x // a="F", e={b: "M", c: 2, d: "R"}


.join(', '): Transforma a texto y une los elementos de una lista con una coma y espacio (', ' -> pegamento)

Destructuracion (extraer propiedades) de objetos
const class = {a:'foo', b:'var'} 
a. Desestructurar objeto -> const {a,b} = class; //a='foo', b='var'
a. Renombrar variables -> const {a:foo, b:var} = class; // foo='foo', var='var'
b. Valores x defecto(evitar undefined) -> const {a,b,c='baz'} = class; // a='foo', b='var', c='baz'
c. Desestructuracion anidada -> class2 = {a:'foo', b:{c:'baz'}} -> const {b:{c}} = class2 // c='baz'
d. Parametros de funciones: Pasar objeto a funcion y desestructura en parentesis de funcion
function print({a,b}) {console.log(`Hi! ${a}, ${b}`)}
print({a:'foo', b:'var'}) // Hi! foo, var

!!a -> Convierte valor en booleando (true/false)

Funciones normales:
function a() {...code} // Se puede usar antes de declarar, tiene propio this
Funciones arrow:
const a = () => {...code} // No se puede usar antes de declarar, herada this

Funciones pueden retornar objetos
function a() {a='foo'; b='var'; return {b,c}}
const d = a // d.b='foo', d.c='var'
